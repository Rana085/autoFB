​🚀 FB-AUTO-MASTER (Ultimate Multi-Threaded Edition)
​FB-AUTO-MASTER হলো একটি অত্যাধুনিক ফেসবুক অ্যাকাউন্ট অটোমেশন টুল, যা পাইথন দিয়ে তৈরি। এটি মূলত টার্মাক্স (Termux) এবং লিনাক্স ব্যবহারকারীদের জন্য ডিজাইন করা হয়েছে। এতে রয়েছে মাল্টি-থ্রেডিং, অটো-প্রক্সি স্ক্র্যাপিং এবং টেম্পোরারি মেইল ভেরিফিকেশন সিস্টেম।
​✨ বিশেষ বৈশিষ্ট্যসমূহ (Key Features)
​⚡ Multi-Threading: একসাথে একাধিক অ্যাকাউন্ট তৈরির সুবিধা (৫-১০ গুণ দ্রুত)।
​🌐 Auto Proxy Scraper: প্রতি ১০ মিনিট পরপর সচল প্রক্সি অটো-আপডেট।
​📧 Temp-Mail Support: অটোমেটিক ইমেইল জেনারেটর এবং ইনবক্স রিডার।
​🔑 OTP Auto-Fetch: ফেসবুকের ওটিপি কোড সরাসরি স্ক্রিনে প্রদর্শন।
​🛡️ User-Agent Rotation: প্রতিবার নতুন ডিভাইস এবং ব্রাউজার আইডেন্টিটি।
​💾 Auto-Save: সফলভাবে তৈরি হওয়া আইডিগুলো Success_IDs.txt ফাইলে জমা হয়।
​🛠️ ইনস্টলেশন ও ব্যবহার (Installation)
​আপনার টার্মাক্সে নিচের কমান্ডগুলো কপি করে পেস্ট করুন:
# প্যাকেজ আপডেট ও গিট ইন্সটল
pkg update && pkg upgrade -y
pkg install python git -y

# রিপোজিটরি ক্লোন করা
git clone https://github.com/আপনার_ইউজারনেম/FB-AUTO-MASTER
cd FB-AUTO-MASTER

# প্রয়োজনীয় লাইব্রেরি ইন্সটল
pip install -r requirements.txt

# টুলটি চালু করা
python main.py
